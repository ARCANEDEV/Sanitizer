### Requirements
    - PHP >= 5.4.0
    
## Installation

First include the sanitizer package.

    "arcanedev/sanitizer": "dev-master"

Now simply `use` the Sanitizer class.

    use Arcanedev\Sanitizer\Sanitizer;
